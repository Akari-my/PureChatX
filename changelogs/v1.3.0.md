## 📌 Version 1.3.0 – Bug Fixes & Stability Release

### 🐛 Fixed

#### Critical Bugs
- **ChatListener Not Receiving Plugin** (`PCX.php:54`)
  - Fixed bug where `ChatListener` was instantiated without passing the plugin instance
  - Added constructor to ChatListener to properly receive PCX instance

- **Wrong Fallback Group** (`ChatListener.php:17,30`)
  - Fixed default fallback from "default" to "guest" to match PurePermsX default group
  - Now correctly falls back to "guest" when no group is found

- **Null Pointer Exception** (`ChatListener.php:17,30`)
  - Fixed crash when PurePermsX plugin was not loaded
  - Added null check for `PPX::getInstance()` before accessing its methods

- **Redundant Permission Application** (`ChatListener.php:23`)
  - Removed redundant call to `applyPermissions()` in onJoin
  - Permissions are already handled by PurePermsX plugin

- **Duplicate Semicolon** (`PCX.php:53`)
  - Fixed typo with double semicolon `;;` removed

#### ChatFormatManager Fixes
- **Wrong Config Key** (`ChatFormatManager.php:101`)
  - Fixed incorrect config key lookup for nametag format
  - Added multiple fallback keys: "nametag", "nameTag", and default string

- **Missing Tag Validation** (`ChatFormatManager.php:62-74`)
  - Added `isset()` checks in setPrefix, setSuffix, and setFormat methods
  - Prevents creating empty tags when tag doesn't exist

### 📝 Updated
- Improved overall plugin stability
- Better error handling for missing PurePermsX integration
- Optimized ChatListener to use injected plugin instance
